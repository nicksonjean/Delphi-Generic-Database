dbExpress Driver for Oracle
Support

This document describes how you can get help with dbExpress Driver for Oracle, including the Priority Support Program.

Resources and Support Channels

  • Find out more about the dbExpress Driver installation and licensing in the Readme and FAQ documents.
  • Get help from the developer community and technical support on the Devart Forum
  • Get advanced technical assistance from the dbExpress Driver developers through the Priority Support Program.

Subscriptions

The dbExpress Driver Subscription Program is an annual maintenance and support service for dbExpress Driver users.

Users with a valid dbExpress Driver subscription get the following benefits:

  • Technical support through the Priority Support Program
  • Access to new versions of dbExpress Driver when they are released
  • Access to all dbExpress Driver updates and hotfixes
  • Notifications of new product versions

Priority Support

dbExpress Driver priority support is an advanced product support service that provides individual assistance with dbExpress Driver-related questions from the developers. Priority support is provided over email and has a two business day response policy. Priority support is available for users with an active dbExpress Driver subscription.

To get help through the Priority Support Program, please send us an email at dbxoda@devart.com with a description of the issue. Make sure to include the following information:

  • Your version of Delphi or C++Builder.
  • Your dbExpress Driver license number.
  • Your edition and full version number of dbExpress Driver. You can find the version number in the DLL version information.
  • Your versions of Oracle server and client.
  • A detailed description of the issue.
  • If possible, attach a sample project demonstrating the issue. Please include definitions for all database objects and avoid using other third-party components in the sample project.

If you have any questions regarding the licensing or subscriptions, please see the FAQ or contact us at sales@devart.com
